<h1>Poetic Text Generator using LSTM</h1>

<p>A character-level text generation project that uses LSTM neural networks to generate poetic text trained on Shakespeare’s works.</p>

<h2>Overview</h2>
<ul>
  <li>Built with Python using TensorFlow and Keras</li>
  <li>Trained on a subset of Shakespeare’s complete works</li>
  <li>Uses temperature-based sampling for diverse text generation</li>
</ul>

<h2>Features</h2>
<ul>
  <li>Character-level encoding and one-hot vectorization</li>
  <li>Sequential model with LSTM and Dense layers</li>
  <li>Custom text generation function with variable creativity</li>
</ul>

<h2>Results</h2>
<ul>
  <li>Low temperature (0.2): Predictable, repetitive output</li>
  <li>High temperature (0.8): Creative, varied poetic lines</li>
</ul>
